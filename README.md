# gtkshutdown #
A Shutdown Manager based on GTK+

![Screenshot of version 0.1](http://i.imgur.com/8g1m4Mg.png)

## Building ##

#### Compile: ####

$<code>make</code>

#### Run: ####

$<code>make run</code>  or  $<code>cd build; ./gtkshutdown</code>
